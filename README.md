# Dorado División – Sitio Web Corporativo

Sitio web estático para **Dorado División**, empresa exportadora de frutas tropicales colombianas fundada en 1992 en el Valle del Cauca. El sitio integra textos e imágenes de cada categoría de producto en páginas modernas, responsivas y navegables.

## Páginas

| Archivo | Sección |
|---|---|
| `index.html` | Inicio – hero slideshow, ¿Quiénes somos?, tarjetas de categorías |
| `azucar.html` | Azúcar – tipos (blanca, morena, pulverizada, industrial) |
| `catalogo.html` | Catálogo de Frutas – galería con nombre de cada fruta |
| `colaboradores.html` | Colaboradores – tarjetas del equipo |
| `deshidratados.html` | Frutas Deshidratadas – galería de productos |
| `fresca.html` | Fruta Fresca – galería de fruta fresca |
| `pulpa.html` | Pulpa de Fruta Natural – información y galería |

## Abrir localmente

1. Clona o descarga el repositorio.
2. Abre `index.html` directamente en tu navegador (doble clic o arrastra al navegador).
3. Navega entre las secciones usando el menú superior.

> No se requiere servidor ni herramientas adicionales; es HTML/CSS puro.

## Publicar en GitHub Pages

1. Ve a **Settings → Pages** en el repositorio.
2. En *Source*, selecciona la rama `main` y la carpeta `/ (root)`.
3. Haz clic en **Save**.
4. GitHub generará una URL del tipo `https://sarin-jp.github.io/pagina_web_dorado/`.

## Estructura del proyecto

```
index.html          # Página de inicio
azucar.html         # Página de Azúcar
catalogo.html       # Catálogo de Frutas
colaboradores.html  # Colaboradores
deshidratados.html  # Deshidratados
fresca.html         # Fruta Fresca
pulpa.html          # Pulpa de Fruta
styles.css          # Estilos compartidos (responsive, paleta naranja/negro)
contenido/          # Imágenes y textos fuente por categoría
```

## Tecnologías

- HTML5 semántico
- CSS3 (Grid, Flexbox, animaciones, media queries)
- Sin dependencias externas ni pasos de compilación
